

## 浏览器的兼容性问题

IE： 微软的浏览器，随着操作系统安装的。所以每个windows都有IE浏览器。各版本如下：

- windows xp 	操作系统安装的IE6
- windows vista 操作系统安装的IE7
- windows 7 	操作系统安装的IE8
- windows 8 	操作系统安装的IE9
- windows10 	操作系统安装的edge

浏览器兼容问题，要出，就基本上就是出在IE6、7身上，这两个浏览器是非常低级的浏览器。
